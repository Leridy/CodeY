# 博客写作指南

## 命名规则

所有博客文章使用 `[Blog]` 前缀，格式：
```
[Blog] YYYY-MM-DD-<主题>.md
```

示例：
- `[Blog] 2026-07-05-llm-integration.md`
- `[Blog] 2026-07-05-permission-system.md`

## 写作原则

### 1. 减少技术代码
- 少放代码片段
- 用流程图代替代码逻辑
- 用类比解释复杂概念

### 2. 图文并茂
- 使用 ASCII 流程图
- 使用表格对比
- 使用示意图说明

### 3. 便于理解
- 用案例解释复杂逻辑
- 用类比说明技术概念
- 用故事串联知识点

### 4. 结构清晰
- 引言：为什么需要这个功能
- 设计：怎么设计的
- 实现：怎么实现的（少代码）
- 测试：怎么验证的
- 总结：学到什么
